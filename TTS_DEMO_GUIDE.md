# TTS 页面示例展示说明

## 修改内容

在 TTS 生成页面顶部添加了一个**效果示例展示区域**，用于展示语音克隆的效果对比。

## 展示内容

### 示例音频对比
- **原始音频** (`mcy.wav`)：MCY 的原始录音
- **克隆音频** (`mcy-clone.wav`)：使用语音克隆系统生成的音频

### 展示特点
1. **渐变背景卡片**：紫色渐变背景，突出示例区域
2. **左右对比布局**：原始音频和克隆音频并排展示
3. **详细信息**：显示说话人、时长、克隆模型等信息
4. **提示信息**：绿色提示框，引导用户使用

## 文件位置

音频文件已复制到：
- `frontend/public/mcy.wav` - 原始音频
- `frontend/public/mcy-clone.wav` - 克隆音频

这些文件可以通过以下 URL 访问：
- http://localhost:3000/mcy.wav
- http://localhost:3000/mcy-clone.wav

## 视觉效果

### 示例区域
- **标题图标**：播放图标 + "效果示例"
- **背景**：紫色渐变 (#667eea → #764ba2)
- **卡片**：白色背景，圆角设计

### 原始音频卡片
- **标题背景**：浅灰色 (#f0f2f5)
- **内容**：说话人、时长信息

### 克隆音频卡片
- **标题背景**：浅蓝色 (#e6f7ff)
- **标题颜色**：蓝色 (#1890ff)

- **背景**：浅绿色 (#f6ffed)

## 响应式设计

- **桌面端**：两列布局，左右对比
- **移动端**：单列布局，上下排列

## 访问方式

1. 打开浏览器访问：http://localhost:3000
2. 点击顶部导航栏的 "TTS生成"
3. 在页面顶部即可看到示例展示区域
4. 点击播放按钮试听原始音频和克隆音频

## 技术实现

- 使用 Ant Design 的 Card、Row、Col 组件
- HTML5 `<audio>` 标签播放音频
- CSS 渐变背景和响应式布局
- 音频文件放在 `public` 目录，可直接访问

## 注意事项

- 音频文件需要在 `frontend/public` 目录下
- 浏览器需要支持 WAV 格式音频播放
- 音频会自动预加载元数据（preload="metadata"）
